### 함수 

자바스크립트 코드를 작성하다보면 유사하게 동작하는 코드를 작성할때가 있는데 동일하거나 유사한 코드를 필요하다고 

매번 만들다보면 불필요한 중복 코드가 생김. 자바스크립트에서는 공통으로 사용하는 유사 코드를 하나로 묶어 함수라는 이름으로 사용할 수 있다.

#### 함수 선언 

함수는 이름이 붙은 명령들의 모임이라고 말할 수 있으며 자바스크립트에서는 아래와 같이 함수를 만듦

``` javascript
function 함수 이름 (매개변수) {
  함수가 수행하는 명령
}
```

#### 함수 호출 

함수를 선언했다고 해서 함수를 바로 실행할 수 있는 것은 아님. 함수는 이름을 불러 주어야 실행이 되는데 이를 함수 호출이라고 한다.

``` javascript
function greeting() {
  console.log('안녕하세요!')
}

greeting() // 안녕하세요!
``` 

#### 함수의 인수와 매개변수 

자바스크립트에서는 인수와 매개변수라는 기능을 이용해 함수를 호출하면서 값을 주고받음

* `인수` : 함수를 호출하면서 넘겨주는 값

* `매개변수` : 함수에서 넘겨받은 인수를 저장하는 변수

#### 함수 반환

자바스크립트의 함수는 호출에 대한 답으로 값을 반환할 수 있음. 함수에서 값을 반환하려면 return문을 사용.

``` javascript
function getArea(width, height) {
  let area = width * height
  return area
}

let result = getArea(10, 20)
console.log(result) // 200
```

#### 중첩 함수 

자바스크립트는 함수 내에서 또 다른 함수를 선언할 수 있다. 특정 함수 내부에서 선언된 함수를 중첩 함수라고 함.

``` javascript
function greeting() {
  function greetingWithName(name) {
    cosole.log(`hello! ${name}`)
  }

  let name = '이정환'
  greetingWithName(name)
}

greeting() // hello! 이정환 
```

#### 호이스팅 

호이스팅이란 프로그램에서 변수나 함수를 호출하거나 정의하는 코드가 함수 선언보다 위에 있음에도 불구하고, 

마치 선언 코드가 위에 있는 것처럼 동작하는 자바스크립트만의 독톡한 기능

``` javascript
// 함수 func()를 선언을 하기 전에 호출하지만 오류가 발생하지 않는다.
// func()에 대한 선언이 호출 코드보다 아래에 있지만, 마치 호출보다 먼저 함수를 선언한 것처럼 자연스럽게 동작.
func()

function func() {
  console.log("hello")
}
```

#### 함수 표현식 

함수를 생성하고 변수에 값으로 저장하는 방법

함수를 숫자나 문자열처럼 값으로 취급하기 때문에 변수에 함수를 저장할 수 있다.

변수에 함수를 저장하면 변수 이름으로 호출이 가능.

``` javascript
let greeting = function() {
  console.log("hello")
}

greeting() // hello
```

``` javascript
// 이름을 정의하지 않은 함수를 익명 함수라고 부름.
function greetFunc() {
  console.log("hello")
}

greetFunc() // hello

let greeting = grettingFunc // 변수 greeting에 함수 greetFunc를 저장

greeting() // hello
```

#### 콜백 함수 

다른 함수의 매개변수로 전달되어, 특정 작업이 끝난 후 호출되는 함수를 의미.

``` javascript
let yahoo = function() {
  console.log('야호')
}
// setTimeout (함수, 지연시간ms == 1/1000초)
setTimeout(yahoo, 3000) // 3초후에 yahoo 함수 실행
```

#### setTimeout 

* 양식 : setTimeout (함수, 시간)

* 일정 시간 이후에 함수를 실행해 준다.

* 리턴값 : 고유 번호(예약 번호)

* 시간은 ms단위 (1/1000 초)

#### clearTimeout 

* 고유 번호에 해당하는 setTime을 취소한다.

#### setInterval

* 양식 : setInterval(함수, 실행)

* 일정 시산마다 함수를 실행해준다.

* 시간은 ms 단위(1/1000초)

* 리턴값 : 고유 번호(예약 번호)

#### clearInterval

* 고유 번호에 해당하는 setInterval을 취소한다.

#### 화살표 함수 (arrow function)

익명 함수를 매우 간결하게 작성할 때 사용하는 함수 표현식의 단축문법

* 일반적인 익명 함수

``` javascript
const fn1 = function(a, b) {
  console.log(a,b)
  return a,b
}
fn(1,2)
```

* 전달 인자가 하나인 경우에는, 전달 인자의 괄호를 생략할 수 있다.

``` javascript
const fn2 = a => {
  console.log(a)
  return a
}
```

* 전달 인자가 없는 경우엔 괄호를 생략할 수 없다.

``` javascript
const fn3 = () => {
  console.log(a)
  retrun a
}
```

* 함수의 실행문 안에 return만 있는 경우 중괄호, return을 생략할 수 있다.

* 중괄호를 생략한 경우 return도 생략해야한다.

``` javascript
const fn4 = (a,b) => a+b

const fn5 = a => a + 100 // 전달 인자가 하나만 있을경우, 전달 인자의 괄호를 생략할 수 있다. 
```
