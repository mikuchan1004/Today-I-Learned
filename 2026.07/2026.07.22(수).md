## 자바스크립트 JSON , Ajax

### JSON (JavaScript Object Notation)

JSON은 자바스크립트 객체처렴 생겼지만, 사실은 그저 문자열일 뿐이라는걸 명심하자.

- JSON 선언 방법

``` javascript
let json = {}
json = {
  "key": "value", 
  "num" : 1234, 
  "fn" : function() {
    console.log(1)
  },
  "k1" : {
  "k1-1" : 1,
  "k1-2" : 2,
  k:3
  },
  k2 : 'k3'
  }
  console.log(json)
  console.log("" + json)
```

- 값에 접근하는 방법

``` javascript
// 값에 접근하는 방법
// 방법1 : 배열처럼 
console.log("json['key'] :" , json['key'])
console.log("json['num'] :" , json['num'])
console.log("json['k1']['k1-1'] :" , json['k1']['k1-1'])
        
// 방법2 : 프로퍼티처럼 
console.log('json.key :' , json.key)
// console.log('json.k1.k1-2 :' , json.k1.k-2)
console.log('json.k1.k :' , json.k1.k)
```

- 값을 넣는 방법

``` javascript
// 값 넣기
// key가 있으면 덮어쓰기
// key가 없으면 추가
json.num = 456
console.log(json.num)
console.log(json.num2) //undefined (상태값. 오류 아님!)
json.num2 = 222
console.log(json)
```

- 종류

  1. `JSON.parse(문자열)` : 서버에서 가져온 텍스트 (JSON)를 자바스크립트가 읽을 수 있는 '객체'로 변환시켜준다.

  - 예시 코드 

  ``` javascript
  // parse : 문자열을 다시 json으로 만들어 준다
  const json2 = JSON.parse(str)
  console.log(json2)
  ```

  2. `JSON.stringify(객체)` : 자바스크립트 객체를 서버로 보내기 위해 텍스트(JSON)로 포장한다.

  - 예시 코드

  ``` javascript
  // 네트워크 통신할 떄 무조건 문자로 전송한다
  // stringify : json을 문자열로 만들어 준다
  // 단, 값이 함수인 것은 제외 
  const str = JSON.stringify(json)
  console.log(str)
  ```

### Ajax 

화면은 그대로 둔 채로 뒤에서 쓱싹 서버와 소통하는 기술 

- 종류

  `XMLHttpRequest` : 코드가 길고, `onload` , `open` , `send`등등 직접 다 챙겨줘야 해서 복잡하다.

  옛날 브라우저까지 다 지원한다는게 유일한 장점.

- 예시코드 (날씨 예보 API)

``` javascript
  
  window.addEventListener('load', bind)
            
  // 1단계. 먼저 예보 데이터를 받아오기
  function bind() {
    const btn1 = document.querySelector('#btn1')
    btn1.addEventListener('click' , function(){

    // API 키 준비 
    const key = '655611af65cce5c9d9eac89f849a00bb62592dfeef095490fbf938462cf0440e'
                    
    // ajax 객체 생성 
    const xhr = new XMLHttpRequest()
      let url = 'http://apis.data.go.kr/1360000/VilageFcstInfoService_2.0/getUltraSrtFcst'
      url += '?'
      url += 'serviceKey='+key
      url += '&numOfRows=1000'
      url += '&pageNo=1'
      url += '&dataType=JSON'
      url += '&base_date=20260722'
      url += '&base_time=1500'
      url += '&nx=63'
      url += '&ny=110'
            
    // 보낼 준비 
    xhr.open('GET' , url)
                    
    // 보내기
    xhr.send()

    // 결과 확인 
    xhr.onload = function() {
      const data = JSON.parse(xhr.responseText)
      console.log('예보 데이터 받아오기 OK')
      console.log(data)

      console.log(data.response.body.items.item[0].category)
      console.log(data.response.body.items.item[0].fcstValue)
      console.log(data.response.body.items.item[0].fcstTime)

      // 2단계. 카테고리 필터링 
      let item = data.response.body.items.item
      let filtered = item.filter(function(data){
      if(data.category == 'T1H' || data.category == 'RN1' || data.category == 'REH') {
        return true
      }
    })
    // 카테고리 필터링 결과 출력
    console.log(filtered)

    // 필터링된 결과를 보여줄 테이블 가져오기
    const tbody = document.querySelector('#weather')
                        
    // 배열을 순회하면서 HTML 문자열 생성 
    const rows = filtered.map(filtered => `
      <tr>
        <td>${filtered.category}</td>
        <td>${filtered.fcstTime}</td>
        <td>${filtered.fcstValue}</td>
    `).join('')
              
    tbody.innerHTML = rows
    }
  })
}
```
