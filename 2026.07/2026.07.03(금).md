## CSS float

### width 속성 

이 속성의 기본값은 100%이므로, width 속성값을 지정하지 않은 블록 요소는 부모 요소의 가로폭을 가득 채운다.

width 속성을 선언하지 않은 블록 레벨 요소에 float 속성이 선언되면 width가 inline 요소와 같이 content에 맞게 

최소화되고 다음 요소 위에 떠있게 된다.

### float 속성 관련 문제 해결 방법 (이 부분은 구글 제미나이의 도움을 받아 작성하였습니다)

1. float를 쓰면 발생하는 부작용 : 자식이 공중에 떠서 부모가 찌그러지거나, 다음 박스가 밑으로 기어들어간다.

2. 부모가 찌그러질 때 해결책 : 부모 요소의 CSS에 overflow:hidden을 적거나, clearfix:after 를 사용한다.
``` css
.clearfix:after {
    content: "";
    display: block;
    clear: both;
}
```
3. 다음 박스가 파고들 때 해결책 : 파고드는 그 자식 요소의 CSS에 clear:both를 적어 아래로 밀어낸다.

### CSS shadow 

텍스트나 요소에 그림자 효과를 부여하기 위한 속성

1. text-shadow

텍스트에 그림자 효과를 부여하는 속성이다.

``` css
.red {
  text-shadow: 10px 3px red;
}
```

``` css
.blur {
    text-shadow: 1px 1px 2px red;
    color:white;
    font-size: 2em;
}
```

2. box-shadow

요소에 그림자 효과를 부여하는 속성이다.

``` css
.box {
    border: 1px solid red;
    width: 100px;
    box-shadow: 4px 4px 6px grey;
}
```
