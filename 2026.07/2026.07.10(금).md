### 배열

배열은 순서가 있는 요소의 집합이자 여러 개의 항목을 담는 리스트. 

- 배열 선언

  1. 배열 선언법 1

  ``` javascript
  let arrA = new Array();
  ```

  2. 배열 선언법 2

  ``` javascript
  let arrB = [];
  ```

  * 두 번째 방법을 배열 리터럴이라고 하며 문법이 간결하기 때문에 특별한 이유가 없으면 이 방법으로 배열을 생성.

  * 배열을 생성하면서 값도 할당하고 싶으면 대괄호 안에서 콤마로 값을 구분해 입력하면 됨
 
  ``` javascript
  let food = ['짜장면' , '피자', '치킨']
  ```

  * 배열의 값으로 어떤 자료형도 사용할 수 있다. 배열은 다른 배열은 물론 객체나 함수 등도 모두 저장됨.
 
  * 자바스크립트의 배열은 다른 언어와는 다르게 길이가 고정되어 있지 않는다.

- 배열 인덱스

  - 인덱스란 배열 요소의 위치를 0부터 시작하는 숫자로 순서대로 표현한 것.
 
  - 배열은 인덱스 기능을 이용해 배열의 요소를 순서대로 번호를 매긴다
 
  - 인덱스를 이용하면 배열 요소를 수정하거나 추가할 수 있다.
