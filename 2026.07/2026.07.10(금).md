### 배열

배열은 순서가 있는 요소의 집합이자 여러 개의 항목을 담는 리스트. 

- **배열 선언**

  1. 배열 선언법 1
     ```javascript
     let arrA = new Array();
     ```

  2. 배열 선언법 2
     ```javascript
     let arrB = [];
     ```
     * 두 번째 방법을 배열 리터럴이라고 하며 문법이 간결하기 때문에 특별한 이유가 없으면 이 방법으로 배열을 생성.
     * 배열을 생성하면서 값도 할당하고 싶으면 대괄호 안에서 콤마로 값을 구분해 입력하면 됨.
       ```javascript
       let food = ['짜장면' , '피자', '치킨']
       ```
     * 배열의 값으로 어떤 자료형도 사용할 수 있다. 배열은 다른 배열은 물론 객체나 함수 등도 모두 저장됨.
     * 자바스크립트의 배열은 다른 언어와는 다르게 길이가 고정되어 있지 않음.

- **배열 인덱스**

  - 인덱스란 배열 요소의 위치를 0부터 시작하는 숫자로 순서대로 표현한 것.
  - 배열은 인덱스 기능을 이용해 배열의 요소를 순서대로 번호를 매김.
  - 인덱스를 이용하면 배열 요소를 수정하거나 추가할 수 있음.

- **배열 요소의 추가 및 제거 메서드** (배열 요소는 배열에 담긴 하나의 값)

  1. `push` : 배열의 맨 끝에 요소를 추가하고 새로운 길이를 반환하는 메서드
     ```javascript
     let food = ['짜장면' , '피자' , '치킨']
     const newLength = food.push('탕수육')

     console.log(food) // ['짜장면' , '피자' , '치킨' , '탕수육']
     console.log(`새로운 배열의 길이: ${newLength}`) // 새로운 배열의 길이 : 4
     ```
     * `push` 메서드로 여러 요소를 추가하려면 콤마로 구분해 전달하면 됨.

  2. `pop` : 배열의 맨 끝 요소를 제거하고 제거한 요소를 반환하는 메서드.
     ```javascript
     let food = ['짜장면' , '피자' , '치킨']
     const removedItem = food.pop()

     console.log(removedItem) // 치킨
     console.log(food) // ['짜장면', '피자']
     ```

  3. `shift` : `pop` 메서드와 반대로 배열의 맨 앞 요소를 제거하고 제거한 요소를 반환하는 메서드.
     ```javascript
     let food = ['짜장면' , '피자' , '치킨']
     const removedItem = food.shift()

     console.log(removedItem) // '짜장면'
     console.log(food) // ['피자', '치킨']
     ```

  4. `unshift` : `push` 메서드와 반대로 배열 맨 앞에 요소를 추가하고 새 배열의 길이를 반환하는 메서드.
     ```javascript
     let food = ['짜장면' , '피자' , '치킨']
     const newLength = food.unshift('갈비찜') 

     console.log(food) // ['갈비찜' , '짜장면' , '피자' , '치킨']
     console.log(`새로운 배열의 길이 : ${newLength}`) // 새로운 배열의 길이 : 4
     ```

  5. `slice` : 마치 가위처럼 기존 배열에서 특정 범위를 잘라 새로운 배열을 반환. 원본 배열은 수정되지 않음.
     * 사용법
       ```javascript
       arr.slice(start, end) // start는 잘라낼 범위의 시작, end는 잘라낼 범위의 끝을 지정하는 인덱스
       ```
     * 예시
       ```javascript
       const arr = [1, 2, 3]
       const sliced = arr.slice(0, 2)

       console.log(arr) // [1, 2, 3]
       console.log(sliced) // [1, 2]
       ```
     * `start`만 적고 `end`를 적지 않으면 `start`부터 배열 끝까지 잘라낸 새 배열을 반환.
       ```javascript
       const arr = [1, 2, 3]
       const sliced = arr.slice(2)

       console.log(sliced) // [3]
       ```
     * 음수값을 인덱스로 전달해도 됨. 만약 `end` 없이 `start`만 음수 인덱스로 전달하면 배열 맨 끝부터 전달할 음수의 절대값만큼 잘라낸 새 배열을 반환함.
       ```javascript
       const arr = [1, 2, 3, 4, 5]

       console.log(arr.slice(-1)) // [5]
       console.log(arr.slice(-2)) // [4, 5]
       console.log(arr.slice(-3)) // [3, 4, 5]
       console.log(arr.slice(-4)) // [2, 3, 4, 5]
       console.log(arr.slice(-5)) // [1, 2, 3, 4, 5]
       ```

- **탐색 메서드** : 배열에서 특정 요소를 검색할 수 있음.

  1. `indexOf` : 배열에서 찾으려는 요소의 인덱스를 반환하는 메서드.
     * 사용법
       ```javascript
       arr.indexOf(item, fromIndex) // item : 찾으려는 요솟값, fromIndex : 탐색을 시작할 인덱스 번호
       ```
     * 예시
       ```javascript
       let arr = [1, 3, 5, 7, 1]
       console.log(arr.indexOf(1, 0)) // 0 
       ```
     * 두 번째 인수인 `fromIndex`는 생략이 가능하며, 생략하면 배열의 0번째 인덱스부터 탐색. 또한 `fromIndex`의 값을 음수로 지정할 수 있는데 그러면 탐색 위치는 배열의 맨 뒤에서부터 탐색을 시작함.
       ```javascript
       let arr = [1, 3, 5, 7, 1]

       console.log(arr.indexOf(1)) // 0
       console.log(arr.indexOf(1, -1)) // 4
       ```
     * 찾으려는 요소가 없다면 `-1`을 반환.
       ```javascript
       let arr = [1, 3, 5, 7, 1]
       console.log(arr.indexOf(4)) // -1
       ```

- **변형 메서드** : 배열을 변형하거나 요소를 재정렬하는 메서드

  1. `sort` : 배열의 요소를 정렬할 때 사용하는 메서드
     * 사용법
       ```javascript
       arr.sort(compare(a, b))
       ```
     * 비교 함수로 `compare`를 사용하는데, 필수 사항은 아님. 생략하면 사전순, 오름차순으로 정렬.
       ```javascript
       let arr = ['b' , 'a' , 'c']
       arr.sort()
       console.log(arr) // ['a' , 'b' , 'c']
       ```
       ```javascript
       let arr = [10, 5, 3]
       arr.sort()
       console.log(arr) // [10, 3, 5]
       ```
     * 이런 결과가 나오는 이유는, `sort` 메서드가 기본적으로 요소를 문자열로 취급해 사전순으로 정렬하기 때문.

  2. `join` : 배열 요소를 모두 연결해 하나의 문자열로 반환하는 메서드
     * 사용법
       ```javascript
       arr.join(separator) // 분리 기호로 사용되는 구분자를 인수로 전달. 필수 사항은 아님
       ```
     * 구분자는 배열 요소를 합칠 때 각각의 요소를 구분하는 문자열이며, 생략하면 콤마(`,`)를 기본값으로 제공.
       ```javascript
       let arr = ['안녕' , '나는' , '이정환']

       console.log(arr.join()) // 안녕,나는,이정환 
       console.log(arr.join('-')) // 안녕-나는-이정환
       ```

### Date : 자바스크립트에서 날짜와 시간을 다루는 객체 

1. **기본 사용법**
   * 현재 시간의 달력을 새로 생성할 때는 `new Date()`를 사용.

2. **원하는 정보만 꺼낼 때 쓰는 메서드들**
   * **`now.getFullYear()`** : 현재 연도를 꺼내줌 / 주의: `getYear()`는 옛날 방식이니 쓰지 말 것
   * **`now.getMonth()`** : 현재 몇 월인지 꺼내줌 / 주의: 0부터 시작하므로 7월은 6이 나오니까 +1을 꼭 넣어줄 것
   * **`now.getDate()`** : 현재 며칠인지 꺼내줌 / 주의: 요일(Day)과 헷갈리지 말 것
   * **`now.getDay()`** : 현재 무슨 요일인지 꺼내줌 / 주의: 일요일(0)부터 토요일(6)까지 숫자로 반환됨 
   * **`now.getHours()`** : 현재 몇 시인지 꺼내줌 / 주의: 오후 4시는 16시로 표시됨
