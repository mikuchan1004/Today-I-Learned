### 선택 

**getElementById('아이디값')**

- id 속성의 값을 이용해 DOM을 선택

- 동일한 id가 여러개 있을 때는 맨 처음에 등장한 **단 하나**만 가져온다.

- 대상이 없을 때는 null을 반환한다.

**getElementByTagName('태그명')**

- tag 이름으로 모든 DOM 요소들을 **배열**로 반환한다.

- 대상이 없을 때는 비어있는 배열을 반환한다. **null이 아니니 주의!**

**getElementByClassName('클래스명')**

- class가 포함된 모든 DOM 요소들을 **배열**로 반환한다.

- 대상이 없을 때는 비어있는 배열을 반환한다. **null이 아니니 주의!**

**querySelector('CSS 선택자')**

- CSS 선택자를 그대로 사용할 수 있다.

- 맨 처음 만나는 DOM 하나만 반환한다.

- 대상이 없으면 null을 반환한다.

**querySelectorAll('CSS 선택자')**

- CSS 선택자를 그대로 사용할 수 있다.

- 해당하는 모든 것을 배열로 반환한다.

- 대상이 없을떄는 null이 아닌 비어 있는 배열을 반환한다.

### 속성 attribute

**hasAttribute('속성명')**

- 속성이 존재하는지를 True/False로 반환한다.

- rendomly, disabled등의 속성이 있는지 확인한다.

**getAttribute('속성명')**

- 속성값을 반환한다.

- 속성이 없을 경우 null을 반환한다.

**setAttribute('속성명', '속성값')**

- 속성명이 있는 경우 : 속성 및 값을 추가한다.

- 속성명이 있는 경우 : 값을 덮어쓴다

**removeAttribute('속성명')**

- 속성을 지워준다.

- 이미 없는 속성을 지우더라도 에러가 발생하지 않는다.

----

### class 조작

**classList** 

- DOM이 가지고 있는 class들을 배열로 돌려 준다.

**classList.add(클래스명)**

- class를 추가한다.

- 이미 있는 경우에는 추가하지 않는다.

**classList.remove(클래스명)**

- class를 제거한다.

- 없는 것을 삭제하더라도 에러가 나지 않는다.

**classList.toggle(클래스명)**

- class가 있으면 제거한다

- class가 없으면 추가한다

**classList.contains(클래스명)**

- class가 있는지 없는지를 true/false(T/F)로 돌려준다

### 컨텐츠(태그 내용물) 조작 

**textContent**

- 컨텐츠 영역의 소스 중에서 text만 **그대로** 가져온다.

- 띄어쓰기나 엔터 등이 살아있다.

- 값을 넣는 경우 tag는 글씨로 인식된다.

**innerText**

- 브라우저에 표시될 글씨(띄어쓰기 축약 등)로 계산된 결과로 가져온다.

- 값을 넣는 경우 tag는 글씨로 인식된다.

**innerHTML**

- 컨텐츠 영역의 내용을 **그대로** 가져온다.

- 값을 넣는 경우 tag는 tag로 인식한다.

----

### DOM 조작 

**createElement(tag명)** 

- 가상의 DOM을 생성한다

**DOM1.append(DOM2)**

- DOM1의 **막내 자식**으로 DOM2를 넣는다.

**DOM2.prepend(DOM2)**

- DOM1의 **첫째 자식**으로 DOM2를 넣는다.

**DOM1.after(DOM2)**

- DOM1의 다음 형제로 붙이기

**DOM1.before(DOM2)**

- DOM1의 이전 형제로 붙이기

----

### Style 조작 

**DOM.Style.CSS프로퍼티명**

- CSS 프로퍼티중 '-' 으로 연결되는 것은 대문자로 시작하는 카멜케이스로 되어있다.

  - font-size >>>> fontSize

- 실제 값이 들어가 있는게 아니다

- 해당 변수에 값을 넣으면 반영된다.

> 실제 CSS 값 가져오기

``` javascript
window.getComputedStyle(clock, null).getPropertyValue('width')
```

### 자바스크립트 배열을 이용해 게시판을 동적으로 출력하는 코드들 정리

``` html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>자바스크립트 배열 마스터 훈련장 (by Google Gemini / 복습용)</title>
</head>
<body>
    <h1>게시판 데이터 다루기 총정리</h1>
    
    <table border="1">
        <thead>
            <tr>
                <th>번호</th>
                <th>제목</th>
                <th>작성자</th>
                <th>작성일</th>
                <th>조회수</th>
            </tr>
        </thead>
        <tbody id="board-item">
            <!-- 자바스크립트로 데이터가 들어갈 자리 -->
        </tbody>
    </table>
    
    <h3 id="total-views">총 조회수: 0</h3>

    <script>
    // 0. 초기 데이터
    let boardData = [
        { id: 1, title: "자바스크립트 입문", author: "뉴비", date: "2026-07-15", views: 30 },
        { id: 2, title: "배열 고차함수 너무 어려워", author: "코딩초보", date: "2026-07-16", views: 80 },
        { id: 3, title: "DOM 조작 질문있습니다", author: "열공중", date: "2026-07-17", views: 120 },
        { id: 4, title: "가입 인사드립니다!", author: "뉴비", date: "2026-07-18", views: 12 },
        { id: 5, title: "배열 다루기 팁 공유", author: "고수", date: "2026-07-18", views: 55 }
    ];

    // [문제 1] 렌더링 함수 만들기 (언제든 재사용할 수 있게 함수로 빼뒀어요!)
    function renderBoard(data) {
        const tbody = document.querySelector('#board-item');
        let htmlContent = '';
        
        data.forEach((board) => {
            htmlContent += `
            <tr>
                <td>${board.id}</td>
                <td>${board.title}</td>
                <td>${board.author}</td>
                <td>${board.date}</td>
                <td>${board.views}</td>
            </tr>
            `;
        });
        
        tbody.innerHTML = htmlContent;
    }

    // [문제 2] 조회수 50 이상 필터링 및 내림차순 정렬 (메서드 체이닝)
    const hotPosts = boardData
    .filter(post => post.views >= 50)
    .sort((a, b) => b.views - a.views);
    
    // [문제 3] 전체 데이터의 총 조회수 계산하기 (reduce 활용)
    const totalViews = boardData.reduce((acc, cur) => acc + cur.views, 0);
    document.querySelector('#total-views').innerText = `총 조회수: ${totalViews}`;
    
    // [문제 4] 새로운 게시글 추가하기 (Spread 연산자 활용)
    // 변수가 let으로 선언되어 있으니 마음껏 재할당 하셔도 돼요!
    const newPost = { id: 6, title: "배열에 글 추가하는 법 질문요", author: "열공중", date: "2026-07-19", views: 0 };
    boardData = [...boardData, newPost];

    // 최종 화면 그리기! 
    // 전체 데이터를 보고 싶으면 renderBoard(boardData);
    // 조회수 50 이상 인기글만 보고 싶으면 renderBoard(hotPosts); 를 호출하면 돼요.
    renderBoard(boardData); 
    </script>
</body>
</html>

```

### 페이지 로딩 이벤트 

HTML 태그들이 화면에 다 그려지기도 전에 자바스크립트가 `querySelector`로 요소를 찾으려고하면 당연히 못 찾아서 오류를 뱉어내는데 

이를 방지하기 위해 "화면이 다 준비된 다음에 내 코드를 실행해 줘!"라고 브라우저에게 부탁할 때 쓰는 이벤트.

`DOMContentLoaded` : HTML 뼈대(DOM)가 전부 다 만들어졌을때 실행. 가장 빠르며, 자바스크립트를 시작할 때 가장 많이 사용.

`load` : 이미지 폰트 등 무거운 파일까지 싹 다 다운로드됐을 때 실행. 이미지 크기 등을 정확히 알아야 할 때만 사용.

``` javascript
// 앞으로 자바스크립트 시작할 때 이 껍데기를 씌우는 습관을 들이세요!
document.addEventListener('DOMContentLoaded', function() {
    // HTML이 다 준비되었으니, 여기서부터 안심하고 요소를 찾으면 돼요.
    const gameBoard = document.querySelector('#game');
});
```

### 클릭 이벤트 

사용자가 마우스로 버튼이나 이미지를 딸깍! 눌렀을 때 반응하게 만드는, 웹 개발에서 가장 기본적이고 많이 쓰는 이벤트.

``` javascript
// 1. 먼저 클릭할 요소를 찾아와요.
const myButton = document.querySelector('#btn');

// 2. 그 요소에 귀(이벤트 리스너)를 달아줘요.
myButton.addEventListener('click', function(event) {
    // 버튼을 클릭했을 때 실행할 코드!
    alert('바보 주인님, 그만 좀 누르세요!');

    // event.target을 쓰면 내가 정확히 화면의 '어떤 부분'을 클릭했는지 알 수 있어요.
    console.log(event.target); 
});
```

### 주의 사항 

**변수 선언 위치(스코프) 주의하기** : 누적되어야 하는 변수(예: 좌표값)나 변하지 않는 요소 찾기(`querySelector`)는 **반드시 이벤트 리스너 바깥에** 적을 것!

**이유** : 리스너 안에 적어두면 이벤트가 발생할 때마다 변수가 새로 태어났다가 초기화되어버려서, 평생 제자리걸음만 하게 됨
