### CSS 속성을 자바스크립트를 사용해서 넣어보자

**CSS에서 -로 연결되는 프로터티명은 카멜케이스(단어를 대문자로 연결)로 변경해서 사용**

**font-size >>> fontSize**

**background-color >>> backgroundColor**

``` javascript
const clock = document.getElementById('clock')
// CSS에서 -로 연결되는 프로퍼티명은 카멜케이스(단어를 대문자로 연결 )로 변경해서 사용
// font-size >>> fontSIze
// background-color >>> backgroundColor
clock.style.color = 'red'
clock.style.fontSize = '1.5em'

console.log(clock.style)
console.log(clock.style.color)
console.log(clock.style.height)
clock.style.height = '50px'
console.log(clock.style.height)
clock.style.border = '1px solid red'
```
