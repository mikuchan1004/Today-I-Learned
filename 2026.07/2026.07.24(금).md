## 파이썬 기본 문법, 변수와 연산자, 문자열, 배열과 튜플

### 파이썬

1991년 발표된 고급 프로그래밍 언어로 문법이 매우 간결하고 직관적.

웹 개발, 데이터 분석, 인공지능(AI), 기계학습 등 다양한 분야에서 널리 사용됨.

#### 기본 문법 

- 세미클론 : 많은 프로그래밍 언어들은 구문이 끝날 때 ;(세미클론)을 붙여야 하지만, 파이썬은 세미클론을 붙이지 않는다.

``` python
print('hello, world!')
```

단, 세미클론을 붙여도 문법 에러는 발생하지 않는다. 보통 한 줄에 여러 문장을 사용할 때 세미클론으로 구분해준다.

``` python
print('hello'); print('1234')
```

- 주석 : 파이썬에서 사람만 알아볼 수 있도록 작성하는 부분을 주석(comment)라고 한다.

즉, 주석은 파이썬 인터프리터가 처리하지 않으므로 프로그램의 실행에는 영향을 주지 않는다.

  - 한 줄 주석 : 코드 맨 앞에 #을 사용하면 해당 줄은 모두 주석이 되어 동작하지 않는다.

  ``` python
  #print('hello world')
  ```

  코드 뒤에 #으로 주석으로 작성할 수도 있다. 이때는 앞에 있는 코드만 정상적으로 동작하며 # 뒤에 있는 코드는 동작하지 않는다.

  ``` python
  a = 1 + 2 # 더하기
  print('Hello, world!') #printf('1234567890')
  ```

  - 블록 주석 : 각 줄마다 맨 앞에 #을 넣어준다.

  ``` python
  #print('Hello, world!')
  #print('1234567890')
  ```

  보통 블록 주석을 작성할 때는 코드를 읽기 쉽도록 # 뒤에 공백을 한 칸 띄운다.

  ``` python
  # 더하기
  # a = 1 + 2
  # print('Hello, world!')
  ```

- 들여쓰기 : 코드를 읽기 쉽도록 일정한 간격을 띄어서 작성하는 방법. 특히 파이썬은 들여쓰기 자체가 문법.

  예를 들어 if의 다음 줄은 항상 들여쓰기를 해야 한다. 만약 들여쓰기를 하지 않으면 문법 에러이므로 코드가 실행되지 않는다.

  ``` python
  if a == 10;
  print('10입니다') # IndentationError : expected an indent
  ```

  올바른 코드는 다음과 같이 if의 다음 줄은 들여쓰기를 해주어야 한다.

  ``` python
  if a == 10;
      print('10입니다')
  ```

- 코드 블록 : 특정한 동작을 위해서 코드가 모여 있는 상태를 뜻하며 파이썬은 들여쓰기를 기준으로 코드 블록을 구성한다.

``` python
if a = 10;
    print('10')
    print('입니다.')
```

단, 같은 블록은 들여쓰기 칸 수가 같아야 하고, 공백과 탭 문자를 섞어 쓰면 안 된다.

#### 변수 : 이름표가 붙은 상자.

``` python
# 변수명 = 데이터
my_age = 25 # 숫자형
my_name = "Alice" # 문자열
```

----

#### 연산자 : 계산과 비교 

**산술 연산자** : `+`(더하기), `-`(뺴기), `*`(곱하기), `/`(나누기), `//`(몫), `%`(나머지)

**비교 연산자** : `==`(같다), `!=`(다르다), `>,<`(크다, 작다)

**논리 연산자** : `and`(둘 다 참), `or`(하나라도 참), `not`(반대)

----

#### 문자열 : 텍스트 다루기

문자열은 결국 '어떻게 자르고 붙이느냐`가 핵심. 

``` python
text = "Hello, Python!"

# 인덱싱 (한 글자만 뽑기)
print(text[0]) # H (파이썬은 0부터 시작!)

# 슬라이싱 (어디에서 어디까지 자르기)
print(text[0:5]) # Hello (5번째 앞인 4번째까지만 잘림)
```

----

#### 리스트와 튜플 

파이썬에서는 배열 대신 주로 리스트를 사용하는데, 리스트와 튜플은 쓰임새가 비슷해서 헷갈리기 쉬우니 주의!

|특징|리스트(List)|튜플(Tuple)|
|:---|:---|:---|  
|**생성 기호**|대괄호 `[]`|소괄호 `()`|
|**수정(변경/추가)**|**가능**|**불가능**|
|**사용 목적**|데이터가 자주 바뀔 때|데이터가 절대 변하면 안 될 때|
|**예시**|`my_list = [1, 2, 3]`|`my_tuple = (1, 2, 3)`|
