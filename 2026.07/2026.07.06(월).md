### 자바스크립트 : 웹페이지를 만들 때 사용하는 언어.

### 변수와 상수 

- 변수 

  - 프로그램을 실행하는 과정에서 변경될 수 있는 값을 저장하는 저장소

  - 변수는 값을 가르키는 이름이면서 동시에 저장소.

  - 변수에 저장된 값은 프로그램을 실행하는 도중에 변경할 수 있다.

  - 변수는 이름으로 구분하기에 let으로 변수를 선언할 경우 이름을 중복해서 사용할 수 없다.

  - var로도 변수를 선언할 수 있는데, 이름을 중복해서 사용할 수 있지만 코드가 많아지고 복잡해질 때 실수할 가능성이 커지기 때문에 주의하자.

- 상수 

  - 변수처럼 이름을 가진 저장 공간.

  - 변수와 다르게 프로그램 실행 과정에서 값을 변경할 수 없다.

### 명명 규칙

변수와 상수는 이름을 갖는다는 공통점이 있다. 자바스크립트에서는 프로그래머가 

변수나 상수의 이름을 자유롭게 정할 수 있지만 반드시 지켜야하는 규칙이 4가지 있다. 

이를 지키지 않을경우 오류가 발생한다.

  1. 기호 사용 규칙 

  - 변수의 이름에는 한글을 포함 문자, 숫자, 그리고 특수 기호의 일부를 사용할 수 있다. 

  - 사용이 가능한 특수 기호는 _(언더바) 와 $(달러)만 허용하며 그 외의 기호는 사용할 수 없다.

  2. 숫자 사용 규칙

  - 변수 이름으로 숫자를 사용할 수 있지만 변수의 이름을 숫자로 시작해서는 안된다. 

  - 굳이 숫자를 앞에 쓸 필요가 없다면 변수의 이름 중간이나 뒤에 배치하는것이 좋다.

  3. 예약어 규칙 

  - 예약어란 예약된 단어를 의미.

  - let, const 키워드처럼 자바스크립트에서 이미 사용하기로 약속한 단어는 변수명으로 쓸 수 없다.

  4. 대소문자 구별 

  - 자바스크립트는 변수의 대소문자를 구분하기 때문에 code와 Code 처럼 혼동할 수 있는 변수명을 사용하는 것은 바람직하지 않다.

  5. 좋은 변수 이름 

  - 변수의 이름은 그 이름만으로도 어떤 역할을 하고, 어떻게 동작할지 유추할 수 있도록 작성하는게 좋다. 

  즉, 다른 사람이나 미래의 내가 보더라도 그 변수가 무슨 역할을 하는지 알아볼 수 있게 변수의 이름을 짓는게 좋다.

### 자료형 : 값을 성질에 따라 분류한 것.

1. 숫자형 

- 자바스크립트에서 숫자형은 수의 종류를 구분하지 않음. 즉, 소수, 음수, 실수와 같은 수를 모두 포함하는 자료형.

- 자바스크립트에서 0으로 어떤 수를 나누면 Infinity 즉, 무한대라는 값이 나온다.

- NaN은 Not a Number라는 뜻으로 표현이 불가능한 숫자형의 결과를 표현할 때 사용. 

예시) 숫자와 문자열을 곱하면 정상적인 연산의 결과물이 나올 수 없으므로 NaN이라는 값을 얻는다.

2. 문자형 

- 사람의 이름과 같은 문자열을 포함하는 자료형.

- 어떤 값이 문자형이라는 것을 알리기 위해서는 큰따옴표 또는 작은따옴표로 감싸야 한다.

- 문자형에 덧셈 연산자를 사용하면 두 개의 문자를 이어 붙인다.

- 백틱을 이용해도 문자열을 만들 수 있다. 이걸 이용하면 문자열 사이에 변수를 사용할 수 있어 값이 변하는 동적인 문자열을 생성할 때 유용.

3. 불리언, null, undefined 

3-1. 불리언 

- 참(True) 또는 거짓(Fales) 만을 저장하는 자료형. 

- 작업을 성공적으로 종료했는지, 어떤 값이 있는지 등을 확인할 때 유용하게 사용.

3-2. null 

- "아무것도 없다"라는 뜻으로 변수에 아무런 값도 할당할 필요가 없을 때 사용.

3-3. undefined 

- 변수를 생성하고 아무런 값도 할당하지 않으면 갖게 됨.

### 형 변환 

어떤 값의 자료형을 다른 자료형으로 변환하는 것.

1. 묵시적 형 변환 

자바스크립트 엔진이 스스로 알아서 변환하는 작업.

``` javascript
let number = 10;
let string = "20";

const result = number + string;
console.log(result); // output : 1020
```

이런 결과가 나온 이유는, 숫자와 문자의 덧셈에서 자바스크립트가 암묵적으로 숫자를 문자로 형 변환해 계산을 했기 떄문.

2. 명시적 형 변환 

묵시적 형 변환과 반대로 내장 함수 등을 이용해 프로그래머가 의도적으로 어떤 자료형을 다른 자료형으로 변경하는 작업.

2-1. 숫자형으로 변환 

``` javascript 
let strA = "10";
let strB = "10개";

let numA = Number(strA);
let numB = Number(strB);
 
console.log(numA); // output : 10
console.log(numB); // output : NaN
```

함수 Number는 자바스크립트가 기본적으로 제공하는 내장 함수로, 제공된 문자열을 숫자로 변환해 반환. 

그러나 변수 strB처럼 숫자가 아닌 문자를 포함한 문자열은 정상적으로 변환되지 않기 때문에 NaN을 반환.

숫자뿐만 아니라 문자도 함께 포함된 문자열을 숫자로 변환하고 싶다면 함수 parseInt를 사용.

``` javascript
let strA = "10";
let strB = "10개";

let numA = parseInt(strA, 10);
let numB - parseInt(strB, 10);

console.log(numA); // output : 10
console.log(numB); // output : 10
```

함수 parseInt는 Number처럼 괄호 안에 있는 문자열을 숫자로 변환하는 자바스크립트의 내장 함수.

Number와 달리 괄호 안에 두 개의 값을 콤마로 구분해 전달을 하는데, 첫 번째 값은 변환하려는 문자열이고 두 번째 값은 십진수.

함수 Number는 숫자가 아닌 문자를 포함한 문자열을 변환할 수 없지만 함수 parseInt는 

문자열에서 숫자만 추려 반환하기 때문에 문자와 숫자가 섞여 있는 문자열도 숫자로 변환할 수 있다.

2-2. 문자형으로 변환

``` javascript
let num = 2022;
let str = String(num);

console.log(str); // output : 2022
```

변수 String도 자바스크립트 내장 함수로 인수로 제공한 값을 문자열로 변환해 반환한다.

2-3. 불리언으로 변환 

불리언 값으로 변환하는 내장 함수로는 Boolean이 있다. 

``` javascript 
let varA = "하이";
let varB = 0;
let varC = "";

let boolA = Boolean(varA);
let boolB = Boolean(varB);
let boolC = Boolean(varC);

console.log(boolA); // output : true
console.log(boolB); // output : false
console.log(boolC); // output : false
```

### 연산자 

+, - , * , / 등과 같이 프로그래밍 언어어세 다영한 연상을 수행할 때 도움을 주는 기호 또는 문자.

1. 대입 연산자 

- 가장 기본이 되는 연산자로서 변수에 값을 할당하는 역할을 한다. 

- 대입 연산자의 왼쪽에는 값을 저장하는 변수를, 오른쪽에는 값이나 또 다른 변수를 지정.

- 대입 연산자를 사용할 때 피연산자의 결합 방향은 오른쪽에서 왼쪽으로 진행되므로 연쇄적인 할당이 가능.

``` javascript
let = numA = 1;
let = numB;
let = numC;

numB = numC = numA; 
console.log(numA, numB, numC); // output : 1 1 1
```

2. 산술 연산자

- 덧셈, 뺄셈, 곱셈, 나눗셈, 나머지 연산처럼 프로그래밍에 필요한 가장 기본적인 기능을 수행하는 연산자

- 곱셈, 나눗셈, 나머지 연산자가 덧셈, 뺄셈 연산자보다 우선순위가 높다.

만약 연산자가 동등한 우선순위를 가질 때는 왼쪽에서 오른쪽으로 차례대로 계산한다.

- 소괄호를 사용하면 원하는 연산부터 먼저 수행할 수 있다.

``` javascript
let numberA = 1 + 2 * 10;

console.log(numberA); // output : 21
```

``` javascript
let numberA = (1 + 2) * 10;

console.log(numberA); // output : 30
```

3. 복합 대입 연산자

- 대입 연산자와 산술 연산자의 기능을 함께 이용할 때 사용하는 연산자.

- 복합 대입 연산자를 쓰지 않고 값이 이미 들어있는 어떤 변수에 10을 더하려면 아래와 같이 코드를 작성해야한다.

``` javascript
let number = 10;
number = number + 10;

console.log(number); // output : 20
```

- 복합 대입 연산자를 사용하면 같은 코드를 좀 더 간단하게 표현할 수 있다.

``` javascript
let number = 10;
number += 10; // 산술 연산자와 대입 연산자를 서로 이어 붙여 사용.

console.log(number); // output : 20
```
- 복합 대입 연산자는 모든 산술 연산자와 함께 쓸 수 있다. 

``` javascript
let number = 10;

number += 10
console.log(number); // output : 20

number -= 10
console.log(number); // output : 10 

number *= 10
console.log(number); // output : 100

number /= 10
console.log(number); // output : 10

number %= 10
console.log(number); // output : 0
```
