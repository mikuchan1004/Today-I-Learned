### 증감 연산자

- 값을 1씩 늘리거나 줄일 때 사용하는 연산자

- 사용할때는 덧셈이나 뺄셈 연산자를 두 개 연달아 붙여서 사용.

``` javascript
let a = 1;
a++; // 증감 연산자를 변수 뒤에 두면 증감 연산의 결과는 연산자가 있는 다음 행부터 반영. (후위 연산)
console.log(a); // output : 2

let b = 1;
b--;
console.log(b); // output : 8
```

``` javascript
let a = 1;
// 증감 연산자를 변수 앞에 두어 증감 연산을 작성한 행부터 적용하는 것을 전위 연산이라고 한다.
console.log(++a); // output : 2 
console.log(a); // output : 2

let b = 1;
console.log(--b); // output : 0
console.log(b); // output : 0
```

### 논리 연산자

- 참과 거짓을 포함하는 불리언 값을 다룰 때 사용하는 연산자이며, 종류는 3가지가 있다.

- 종류

  - OR : 연산에 참여하는 두 개의 피연산자 중 하나라도 참이면 참을 반환. (기호 : ||)

  ``` javascript
  let boolA = true;
  let boolB = false;
  // OR 연산을 할때는 변수 사이에 기호(||)를 넣어서 사용.
  console.log(boolA || boolB); // true
  ```

  - AND : 연산에 참여하는 두 개의 피연산자 중 하나라도 거짓이면 거짓을 반환. (기호 : &&)
 
  ``` javascript
  let boolA = true;
  let boolB = false;
  // AND 연산을 할때는 변수 사이에 기호(&&)를 넣어서 사용.
  console.log(boolA && boolB); // false
  ```

  - NOT : 피연산자의 값이 거짓이면 참, 참이면 거짓으로 바꾸어 반환. (기호 : !)

  ``` javascript
  let boolA = true;
  let boolB = false;
  // NOT 연산을 할때는 변수 앞에 기호(!)를 넣어서 사용
  console.log(!boolA); // false 
  console.log(!boolB); // true
  ```

### 비교 연산자

- 두 값을 비교하는 연산자.

- 종류

  - 같다 : 대입 연산자인 = 기호를 3개 이어 붙인 ===을 사용하며, 두 값이 같은지 확인.

  ``` javascript
  let numberA = 2;
  let numberB = 2;
  let numberC = '2';

  console.log(numberA === numberB); // true
  console.log(numberB === numberC); // false
  console.log(numberB == numberC); // 두 변수의 값을 비교하고 결과를 출력. 결과 : true
  ```
  - 같지 않다 : 두 값이 다른지 비교할때는 !== 연산자를 사용하며, 값과 자료형 중 하나라도 다르면 참으로 판단.

  ``` javascript
  let numberA = 2;
  let numberB = 2;
  let numberC = '2';

  console.log(numberA !== numberB); // false
  console.log(numberB !== numberC); // true
  console.log(numberB != numberC); // 자료형은 비교하지 않고 값이 다르면 참으로 판단. 결과 : false
  ```

  - 대소 비교 : 두 값의 크고 작음을 비교할때는 > 또는 <를 사용.

  대소 관계 연산자 > 또는 <에 대입 연산자 =를 붙여 두 값이 서로 크거나 같다(>=) 혹은 작거나 같다 (<=) 처럼 비교할 수 있다. 

  ``` javascript
  let numberA = 1;
  let numberB = 2;
  let numberC = 3;

  console.log(numberA < numberB); // true
  console.log(numberA > numberB); // false
  console.log(numberB < numberC); // true
  console.log(numberB > numberC); // false
  ```

  ``` javascript
  let numberA = 1;
  let numberB = 2;  
  let numberC = 3;

  console.log(numberA <= numberB); // true
  console.log(numberA >= numberB); // false
  console.log(numberB <= numberC); // true
  console.log(numberB >= numberC); // true
  ```

### 동적 타이핑과 typeof 연산자

- 자바스크립트는 변수에 숫자를 저장했다가 문자로 바꿔도 값을 저장할 때마다 자료형이 동적으로 결정되기 때문에 오류가 발생하지 않는다.

- 이러한 유연함은 단점이 되기도 하는데, 프로그래머가 변수의 자료형이 변경되었다는 사실을 모르고 코드를 작성하면

프로그램이 의도치 않게 작동하거나 오류가 발생할 수 있다. 이런 상황에 대비해 변수의 자료형이 무엇인지 확인할 필요가 있는데 

typeof 연산자를 사용하면 변수의 자료형을 확인할 수 있다. 

``` javascript
let varA = 1;
varA = '이정환';

console.log(typeof VarA); // string
```

### 삼항 조건 연산자 

- 유일하게 자바스크립트에서 3개의 피연산자를 취하는 연산자.

- ? 앞에 조건식을 작성하고, 콜론(:)을 기준으로 앞은 조건식이 참일 때 그리고 뒤는 조건이 거짓일 때 수행할 명령을 작성하면 된다. 

``` javascript
const varA = "안녕하세요";

typeof varA === "string";
? console.log("문자 자료형")
: console.log("문자 자료형이 아님") // 출력 : 문자 자료형
```

1. if문

- "만약 A라면 B를 하고, 그렇지 않으면 C를 하라" 와 같이 조건에 따라 각기 다른 명령을 수행하도록 만들 때 사용

- if 키워드 다음의 소괄호에 조건식을 입력하고, 중괄호에서는 해당 조건식이 참일 때 수행할 명령을 입력.
