### 증감 연산자

- 값을 1씩 늘리거나 줄일 때 사용하는 연산자

- 사용할때는 덧셈이나 뺄셈 연산자를 두 개 연달아 붙여서 사용.

``` javascript
let a = 1;
a++; // 증감 연산자를 변수 뒤에 두면 증감 연산의 결과는 연산자가 있는 다음 행부터 반영. (후위 연산)
console.log(a); // output : 2

let b = 1;
b--;
console.log(b); // output : 8
```

``` javascript
let a = 1;
// 증감 연산자를 변수 앞에 두어 증감 연산을 작성한 행부터 적용하는 것을 전위 연산이라고 한다.
console.log(++a); // output : 2 
console.log(a); // output : 2

let b = 1;
console.log(--b); // output : 0
console.log(b); // output : 0
```

### 논리 연산자

- 참과 거짓을 포함하는 불리언 값을 다룰 때 사용하는 연산자이며, 종류는 3가지가 있다.

- 종류

  - OR : 연산에 참여하는 두 개의 피연산자 중 하나라도 참이면 참을 반환. (기호 : ||)

  ``` javascript
  let boolA = true;
  let boolB = false;
  // OR 연산을 할때는 변수 사이에 기호(||)를 넣어서 사용.
  console.log(boolA || boolB); // true
  ```

  - AND : 연산에 참여하는 두 개의 피연산자 중 하나라도 거짓이면 거짓을 반환. (기호 : &&)
 
  ``` javascript
  let boolA = true;
  let boolB = false;
  // AND 연산을 할때는 변수 사이에 기호(&&)를 넣어서 사용.
  console.log(boolA && boolB); // false
  ```

  - NOT : 피연산자의 값이 거짓이면 참, 참이면 거짓으로 바꾸어 반환. (기호 : !)

  ``` javascript
  let boolA = true;
  let boolB = false;
  // NOT 연산을 할때는 변수 앞에 기호(!)를 넣어서 사용
  console.log(!boolA); // false 
  console.log(!boolB); // true
  ```

### 비교 연산자

- 두 값을 비교하는 연산자.

- 종류

  - 같다 : 대입 연산자인 = 기호를 3개 이어 붙인 ===을 사용하며, 두 값이 같은지 확인.

  ``` javascript
  let numberA = 2;
  let numberB = 2;
  let numberC = '2';

  console.log(numberA === numberB); // true
  console.log(numberB === numberC); // false
  console.log(numberB == numberC); // 두 변수의 값을 비교하고 결과를 출력. 결과 : true
  ```
  - 같지 않다 : 두 값이 다른지 비교할때는 !== 연산자를 사용하며, 값과 자료형 중 하나라도 다르면 참으로 판단.

  ``` javascript
  let numberA = 2;
  let numberB = 2;
  let numberC = '2';

  console.log(numberA !== numberB); // false
  console.log(numberB !== numberC); // true
  console.log(numberB != numberC); // 자료형은 비교하지 않고 값이 다르면 참으로 판단. 결과 : false
  ```

  - 대소 비교 : 두 값의 크고 작음을 비교할때는 > 또는 <를 사용.

  대소 관계 연산자 > 또는 <에 대입 연산자 =를 붙여 두 값이 서로 크거나 같다(>=) 혹은 작거나 같다 (<=) 처럼 비교할 수 있다. 

  ``` javascript
  let numberA = 1;
  let numberB = 2;
  let numberC = 3;

  console.log(numberA < numberB); // true
  console.log(numberA > numberB); // false
  console.log(numberB < numberC); // true
  console.log(numberB > numberC); // false
  ```

  ``` javascript
  let numberA = 1;
  let numberB = 2;  
  let numberC = 3;

  console.log(numberA <= numberB); // true
  console.log(numberA >= numberB); // false
  console.log(numberB <= numberC); // true
  console.log(numberB >= numberC); // true
  ```

### 동적 타이핑과 typeof 연산자

- 자바스크립트는 변수에 숫자를 저장했다가 문자로 바꿔도 값을 저장할 때마다 자료형이 동적으로 결정되기 때문에 오류가 발생하지 않는다.

- 이러한 유연함은 단점이 되기도 하는데, 프로그래머가 변수의 자료형이 변경되었다는 사실을 모르고 코드를 작성하면

프로그램이 의도치 않게 작동하거나 오류가 발생할 수 있다. 이런 상황에 대비해 변수의 자료형이 무엇인지 확인할 필요가 있는데 

typeof 연산자를 사용하면 변수의 자료형을 확인할 수 있다. 

``` javascript
let varA = 1;
varA = '이정환';

console.log(typeof VarA); // string
```

### 삼항 조건 연산자 

- 유일하게 자바스크립트에서 3개의 피연산자를 취하는 연산자.

- ? 앞에 조건식을 작성하고, 콜론(:)을 기준으로 앞은 조건식이 참일 때 그리고 뒤는 조건이 거짓일 때 수행할 명령을 작성하면 된다. 

``` javascript
const varA = "안녕하세요";

typeof varA === "string";
? console.log("문자 자료형")
: console.log("문자 자료형이 아님") // 출력 : 문자 자료형
```

### 조건문 : 특정 조건을 만족할 때 원하는 동작을 수행하게 하는 프로그래밍의 기본 문법 중 하나.

1. if문

- "만약 A라면 B를 하고, 그렇지 않으면 C를 하라" 와 같이 조건에 따라 각기 다른 명령을 수행하도록 만들 때 사용

- if 키워드 다음의 소괄호에 조건식을 입력하고, 중괄호에서는 해당 조건식이 참일 때 수행할 명령을 입력.

``` javascript
let num = 11;
// 변수 num의 값은 11로서 10보다 크기 때문에 중괄호에 작성한 명령을 수행.
if (num >= 10) {
  console.log("num은 10 이상입니다.") // num은 10 이상입니다.
}
```

- if 문에서는 명령을 여러 줄에 걸쳐 입력해도 얼마든지 수행할 수 있다.

``` javascript
let num = 11;

if (num >= 10) {
  console.log("조건 일치")
  console.log("num은 10 이상입니다")
}
```

- 조건식을 만족하지 않을 때 수행할 코드를 따로 작성하고 싶다면 else 키워드를 사용하면 된다.

``` javascript
let num = 9;

if (num >= 10) {
  console.log("조건 일치);
  console.log("num은 10 이상입니다");
}else{
  console.log("조건 불일치");
  console.log("num은 10보다 작습니다");
}
```

- 참과 거짓말고도 또 다른 조건을 추가하려면 else if 키워드를 사용하면 된다.

``` javascript
let num = 5;

if (num >= 10) {
  console.log("num은 10 이상입니다");
} else if (num >= 5) {
  console.log("num은 5 이상입니다");
} else {
  console.log("num은 5 미만입니다");
}
```

- else if 문은 여러 개 중첩해 사용할 수 있어서 조건이 여러 개 있어도 상관없다.

``` javascript
let contury = 'ko';

if (contury === 'ko') {
  console.log('한국');
} else if (contury === 'us') {
  console.log('미국');
} else if (contury === 'dk') {
  console.log('덴마크');
} else {
  console.log('미분류');
```

2. switch문

- 중첩 if문 처럼 비교할 조건이 많을 때 사용하는 조건문. 식이나 값을 case 문과 비교해 정확히 일치할 때만 수행.

``` javascript
lef fruit = 'apple';

switch (fruit) {
    case 'apple' : {
      console.log('사과')
      break;
  }
    case 'banana' : {
      console.log('바나나')
      break;
  }
  default : { 
    console.log('우리가 찾는 과일이 아님') // 출력 : 사과 
  }
} 
```

- switch 문에서는 break 문을 사용한다. case 문에서 수행할 명령을 작성하고 break를 입력해 "실행할 명령이 끝났다"라고 알려줘야함.

break 문을 작성하지 않아도 오류가 발생하지는 않지만 조건식과 일치해 실행한 case 문 이후의 명령까지 모두 차례로 수행한다.

``` javascript
lef fruit = 'apple';

switch (fruit) {
    case 'apple' : {
      console.log('사과')
      
  }
    case 'banana' : {
      console.log('바나나')
      
  }
  default : {
    console.log('우리가 찾는 과일이 아님') // 출럭 : 사과 바나나 우리가 찾는 과일이 아님
  }
} 
```

- 조건이 여러 개인 경우에 switch 문을 사용하면 if 문보다 가독성 있는 코드를 작성할 수 있다.
