## 마우스 이벤트 & 폼 이벤트 

### 마우스 이벤트

마우스 이벤트는 화면의 거의 모든 요소(태그)에 다 달 수 있다. 

#### 이벤트 종류

- `click` / `dbclick` : 클릭, 더블 클릭할 때 발생 / 가장 기본이며, 마우스 왼쪽 버튼 기준

- `mousedown` / `mouseup` : 마우스 버튼을 누르거나 뗄 때 발생 / 클릭보다 더 세밀하게 제어할 수 있으며 드래그 앤 드롭 만들 때 필수

- `mousemove` : 요소 위에서 마우스가 움직일 때 발생 / 움직일 때마다 발생하므로 마우스의 좌표를 추적할 때 사용

- `mouseenter` / `mouseleave` : 마우스가 영역에 들어오거나 나갈 때 발생 / 자식 요소에는 반응을 하지 않아 깔끔하게 쓰기 좋음

- `mouseover` / `mouseout` : 마우스가 영역에 들어오거나 나갈 때 발생 / 자식 요소 위로 마우스가 지나가도 반응.

----

### 폼 이벤트 

폼 이벤트는 사용자가 입력창(`<input>`, `<select>`등)에 무언가를 적거나 선택할 때 사용. **대상 요소가 정해져 있다는 게** 

마우스 이벤트와의 가장 큰 차이점.

#### 이벤트 종류 

- `submit` : 폼(양식) 전체가 전송될 때 발생 / 전송 버튼(`button`)이 아니라 `<form>` **태그 자체**에 달아야한다.

- `focus` / `blur` : 입력창에 커서가 들어오거나 빠져나갈 때 발생 / 입력창 테두리 색깔을 강조하거나 힌트를 띄워줄 때 사용

- `input` : 값을 타건할 때마다 **실시간**으로 발생 / 글자를 칠 때마다 반응하며 실시간 글자 수 세기 같은 데 사용

- `change` : 입력이 **완료**되고 상태가 변했을 때 발생 / 타건 중엔 조용하다가 엔터를 치거나 커서가 밖으로 빠지면 발생한다.

특히 `select` 박스나 `checkbox` , `radio`에서 많이 사용 

---- 

### 이벤트 종류별 예시 코드들 

1. 마우스 이벤트 - `click`/ `dbclick`

``` javascript
area2.addEventListener('click', function(evt){
  log('click')
  log('offsetY :' + evt.offsetY )
  log('pageY :' + evt.pageY )
  log('clientY :' + evt.clientY )
  log('screenY :' + evt.screenY )
})

area2.addEventListener('dbclick' , function(){
  log('더블클릭 발생')
})
```

2. 마우스 이벤트 - `mousedown` / `mouseup`

``` javascript
document.querySelector('#img').addEventListener('mousedown' , function(evt) {
  _isDrag = true
  _offsetX = evt.offsetX
  _offsetY = evt.offsetY
})
document.querySelector('#img').addEventListener('mouseup' , function(evt) {
  _isDrag = false
})
```

3. 마우스 이벤트 - `mousemove`

``` javascript
document.querySelector('body').addEventListener('mousemove' , function(evt){
  const img = document.querySelector('#img')

  if(_isDrag) {
    img.style.top = (evt.pageY - _offsetY) + 'px'
    img.style.left = (evt.pageX - _offsetX) + 'px'
  }
})
```

4. 마우스 이벤트 - `mouseenter`/ `mouseleave`

``` javascript
area2.addEventListener('mouseenter' , function() {
  log('mouseenter')
})

area2.addEventListener('mouseleave' , function() {
  log('mouseleave')
})
```

5. 마우스 이벤트 - `mouseover` / `mouseout`

``` javascript
area2.addEventListener('mouseover' , function(evt) {
  log('mouseover')
  area2.style.backgroundColor = 'yellow'
})

area2.addEventListener('mouseout' , function(evt){
  log('mouseout')
  area2.style.backgroundColor = 'white'
})
```

6. 폼 이벤트 - `submit`

``` javascript
const form = document.querySelector('#form')
form.addEventListener('submit' , function(event){

  // 태그의 기본(고유) 기능을 막아준다 
  event.preventDefault() 

  if(query.value.trim().length < 2) {
    alert('검색어는 두 글자 이상입니다')
  }else{
    form.submit()  
  }
})
```

7. 폼 이벤트 - `focus` / `blur`

``` javascript
query.addEventListener('focus' , function(){
  query.style.background = 'yellow'
})

query.addEventListener('blur' , function(){
  query.style.background = ''
})
```

8. 폼 이벤트 - `input`

``` javascript
// 값이 변경될 때 잡아줌
query.addEventListener('input' , function(){
  log(query.value)

  const r = parseInt(Math.random()*256)
  const g = parseInt(Math.random()*256)
  const b = parseInt(Math.random()*256)
  const a = Math.random()

  query.style.backgroundColor = `rgba(${r}, ${g}, ${b}, ${a})`

})
```

9. 폼 이벤트 - `change`

``` javascript
//전체 선택 체크박스 동작
selectAll.addEventListener('change' , function(){
  const allCheckboxes = document.querySelectorAll('.todo-check')
  allCheckboxes.forEach(cb => {
    cb.checked = selectAll.checked // 모든 체크박스를 전체 선택 체크박스랑 똑같이 맞춤
  })
})
```
