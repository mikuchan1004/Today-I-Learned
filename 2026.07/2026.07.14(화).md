### 자바스크립트 반복문 - for문 (2) , map, sort, filter, string

#### for in

- in 앞의 변수 i에 현재 index를 저장

- 무조건 처음부터 끝까지

``` javascript
let arr = [1,2,3,4,5,6,7]
for(let i in arr) {
  console.log('i :' , i)
  console.log('값 : ' , arr[i])
}
```

#### for of 

- of 앞의 변수에 현재 값을 저장

- 무조건 처음부터 끝까지 반복한다

``` javascript
let arr = [1,2,3,4,5,6,7]
for(let i of arr) {
  console.log('i :' , i)
}
```

#### forEach 

- 모든 요소 순회

- 전달인자 : 현재 값, 현재 index, 배열 전체

- return이 없음.

``` javascript
let r1 = arr.forEach(function (element,  index, array) {
  console.log('element : ' , element) // 현재 값
  console.log('index : ' , index) // 현재 index 값
  console.log('array : ' , array) // 배열 전체
}) 
console.log(r1) // undefined
```

#### sort 

- 자바스크립트에서 배열의 요소를 정렬할 때 사용.

- 기본적으로 요소를 문자열로 변환하여 유니코드 순서대로 정렬하므로, 숫자를 정렬하려면 반드시 비교 함수(콜백)를 인자로 전달해야 한다.

- 정렬을 하면 원본 배열이 변경된다.

``` javascript
r3.sort(function(y, x) {
  return Number(y) - Number(x) // 오름차순
})
console.log(r3)
```

``` javascript
r4.sort(function(y, x) {
  return Number(x) - Number(y) // 내림차순
)}
console.log(r4)
```

#### filter

- 배열의 각 요소를 순회하며 특정 조건을 만족하는 요소들만 추출하여 새로운 배열로 반환하는 메서드

- 원본 배열은 변경되지 않으며 콜백 함수가 true를 반환하는 요소들만 모아줌 

``` javascript
let r6 = movie.filter(function(el, i, a) {
  if(el.length >= 4) {
    return true
  } else {
    return false
  }
})
console.log(r6)
```

#### string 

* `length` : 문자열의 길이

* `indexOf`

  - 동일한 글씨 찾기
 
  - 있으면 동일한 글씨의 첫번쨰 index 반환
 
  - 없으면 -1 반환

* `replace`

  - 양식 : replace("찾을 문자" , "바꿀 문자")
 
  - 왼쪽부터 **딱 하나**의 대상만 변경한다
 
  - 정규표현식으로 전부 바꿀 수 있다.

  ``` javascript
  str.replace(/가/g, '나')
  ```

* `slice`

  - 양식 : slice(시작 index , 끝 index)
 
  - 시작부터 끝 바로 앞까지 복사해서 돌려줌
 

* `split`

  - 구분자로 구분해서 배열로 돌려준다.
 
* `substring`

  - 양식 : substring(시작 index , 끝 index)
 
  - 시작부터 끝 바로 앞까지 복사해서 돌려줌

 * `substr`

   - 양식 : substr(시작 index, 개수)
  
   - 시작부터 몇 개
  
   - 시작은 음수 가능
  
* `trim`

  - 문자열의 앞, 뒤 공백 제거
 
  - 중간의 공백은 제거하지 않는다
