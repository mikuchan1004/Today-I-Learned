### DOM (2)

#### 태그 안쪽의 글씨들 (컨텐츠) 가져오는 방법들 

1. **textContent** : 컨텐츠 영역의 소스 중에서 text(글씨)만 **그대로** 가져온다

   `console.log('msg.textContent' , msg.textContent)`

2. **innerText** : 브라우저에 보이는 글씨로 가져온다. 브라우저가 보여줄 내용에 대한 계산이 끝나고 실제로 표시하는 text

   `console.log('msg.innerText' , msg.innerText)`

3. **innerHTML** : 컨텐츠 영역의 소스를 그대로 가져온다.

   `console.log('msg.innerHTML' , msg.innerHTML)`

> 변수에 값을 담으면 컨텐츠의 내용을 바꿔준다.

> textContent, innerText는 태그도 글씨로 인식한다.

> innerHTML은 태그는 태그로 인식한다.

#### DOM 요소 생성 

``` javaScript
const div = document.createElement('div')
console.log(div)
div.setAttribute('id' , 'lol')
div.innerText = '정글 차이'
console.log(div)

div.setAttribute('style' , 'color:red;')

const log = document.getElementById('log')
```

1. **append** : 마지막 자식 요소로 추가한다. 글씨도 넣을 수는 있지만, 잘 쓰이지는 않는다. 같은 객체는 이동된다.

   `log.append(div)`
   `log.append('<h1>abcd</h1>')`

2. **appendChild** : append처럼 마지막 자식 요소로 추가하는데, 글씨를 넣을 수는 없다.

   `log.appendChild(p)`
   `log.append('<h1>글씨일 뿐이다</h1>') // 이렇게 쓰면 오류 발생`

4. **prepend** : 첫번째 자식 요소로 추가

   `log.prepend(p2)`

5. **before** : 이전 형제로 추가

   `log.before(p3)`

6. **after** : 다음 형제로 추가

   `log.after(p4)`
