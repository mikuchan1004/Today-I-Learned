### while문, do while문 

> 이 내용들은 구글 제미나이의 도움을 받아서 정리했음을 밝힙니다

- while문과 do while문은 둘 다 특정 조건이 맞을 때까지 반복한다는 목적은 같지만 조건을 언제 확인하느냐가 다름.

  - while문

    - 무언가를 실행하기 전에 조건이 맞는지 먼저 깐깐하게 검사를 하는데, 조건이 맞지 않으면 시작조차 안 한다.

    ``` javascript
    while (조건식) {
      // 조건이 참일때 실행할 명령
    }
    ```

  - do...while문 

    - 조건이 맞든 안 맞든 일단 무조건 한 번은 실행하고 봄.

    - 그 다음에 조건을 검사해서 계속할지 말지를 반복.

    ``` javascript {
    do {
      // 실행할 명령
    } while (조건식) 
    ```
