## CSS position 

이 속성은 요소의 위치를 정의한다. top, bottom, left, right 속성과 함께 위치를 지정한다.

### static (기본 위치)

static은 position 속성의 기본값으로 position 속성을 지정하지 않았을 때와 같다. 

기본적인 요소의 배치 순서에 따라 위에서 아래로, 왼쪽에서 오른쪽으로 순서에 따라 배치되며 

부모 요소 내에 자식 요소로서 존재할때는 **부모 요소의 위치를 기준으로 배치된다**

좌표 속성을 같이 사용할 수 없으며 사용할 경우에는 무시된다. 

### relative (상대 위치)

기본 위치를 기준으로 좌표 속성을 사용하여 위치를 이동시킨다.

기본 위치와 상대 위치를 선언한 요소의 차이점은 좌표 속성의 동작 여부뿐이며 그 외에는 동일하게 동작한다. 

``` css
.rel {
  position: relative;     
    /* top: 20px; */      
    top : -20px;          
    left: 10px;           
    /* left: -20px; */
    /* left: 50%; */
}
```
### absolute (절대 위치) 

**부모 요소 또는 가장 가까이 있는 조상 요소 (기본 위치 제외)를 기준으로 좌표 속성만큼 이동한다.**

**부모 또는 조상 요소가 static인 경우, body를 기준으로 하여 좌표 속성대로 위치하게 된다**

부모 요소를 배치의 기준으로 삼기 위해서는 부모 요소에 relative를 정의해야한다. 

``` css
.abs {
  position: absolute;     
    top: 0;             
    left: 0;             
    top: 30px;            
    /* left: 300px; */   
}                         
```
### fixed (고정 위치)

부모 요소와 관계없이 브라우저의 viewport를 기준으로 좌표 속성을 사용하여 위치를 이동시킨다.

**스크롤이 되더라도 화면에 사라지지 않고 항상 같은 곳에 위치한다.**

``` css
.fixed {
    position: fixed;
    right: 10px;
    bottom : 10px;
    border: 1px solid red;
    width: 60px;
    height: 60px;
    text-align: center; 
    line-height: 60px;
}
```
### z-index 속성 

z-index 속성에 큰 숫자값을 지정할수록 화면 전면에 출력되며 position 속성이 static 이외의 요소에만 적용된다.

``` css
.z1 {
  z-index: 99;
}
```
