## CSS position 

이 속성은 요소의 위치를 정의한다. top, bottom, left, right 속성과 함께 위치를 지정한다.

### static (기본값)

static은 position 속성의 기본값으로 position 속성을 지정하지 않았을 때와 같다. 

기본적인 요소의 배치 순서에 따라 위에서 아래로, 왼쪽에서 오른쪽으로 순서에 따라 배치되며 

부모 요소 내에 자식 요소로서 존재할때는 **부모 요소의 위치를 기준으로 배치된다**

좌표 속성을 같이 사용할 수 없으며 사용할 경우에는 무시된다. 

### relative (상대값)

기본 위치를 기준으로 좌표 속성을 사용하여 위치를 이동시킨다.

기본 위치와 상대 위치를 선언한 요소의 차이점은 좌표 속성의 동작 여부뿐이며 그 외에는 동일하게 동작한다. 

``` css
.rel {
  /*
    원래 위치를 기준으로 이동한다.
    내가 가지고 있던 공간이 유지된다.
  */
  position: relative;     
    /* top: 20px; */      
    top : -20px;          
    left: 10px;           
    /* left: -20px; */
    /* left: 50%; */
}
```
### absolute (절대값) 

**부모 요소 또는 가장 가까이 있는 조상 요소 (기본 위치 제외)를 기준으로 좌표 속성만큼 이동한다.**

**부모 또는 조상 요소가 static인 경우, body를 기준으로 하여 좌표 속성대로 위치하게 된다**

새로운 층에 그려진다.
  
  - 브라우저가 표시할 계획 단계에서 빠진다.

  -  그래서 **원래 자리가 없어진다**

  -  inline-block으로 변경 됨.

right, bottom 주의점 

  - 상단 + 좌측 스크롤에서의 viewport의 오른쪽과 아래를 기준으로 그려진다.

  - 그래서 스크롤을 이동해도 거기에 남게 된다.

``` css
.abs {
  /*
    내가 가지고 있던 공간이 없어진다.
    부모중에 relative 등 static이 아닌 것 기준으로 이동
    - 없으면 body 기준
    자동으로 inline-block으로 바뀐다.
  */
  position: absolute;     
    top: 0;             
    left: 0;             
    top: 30px;            
    /* left: 300px; */   
}                         
```
### fixed (고정값)

부모 요소와 관계없이 브라우저의 viewport를 기준으로 좌표 속성을 사용하여 위치를 이동시킨다.

**스크롤이 되더라도 화면에 사라지지 않고 항상 같은 곳에 위치한다.**

``` css
.fixed {
    position: fixed;
    right: 10px;
    bottom : 10px;
    border: 1px solid red;
    width: 60px;
    height: 60px;
    text-align: center; 
    line-height: 60px;
}
```
### sticky 

- 원래 보이는 상태에서는 효과 없음.

- 스크롤을 내려서 안 보이게 될 때 fixed 처럼 동작. 예시) 엑셀의 틀고정

### z-index 속성 

z-index 속성에 큰 숫자값을 지정할수록 화면 전면에 출력되며 position 속성이 static 이외의 요소에만 적용된다.

형제끼리만 적용된다.

- 부모의 z-index를 벗어날 수 없다.

``` css
.z1 {
  /* 기본값은 0 */
  z-index: 99;
}
```

### calc 

- 간단한 계산식이 필요할 때.

- 더하기와 빼기 앞 뒤로 공백 필요

``` css
.calc {
  left: calc (50% - (100px/2));
}
```

### 변수 사용법 : 값에 이름을 붙여놓고 여러군데에서 가져다 사용하는 것.

``` css
:root {
  --box-color : brown;
}

.p1 {
  background-color: var(--box-color);
}

.p2 {
  background-color: var(--box-color);
```

### overflow 속성 : 이 속성은 자식 요소가 부모 요소의 영역을 벗어났을 때 처리 방법을 정의한다.

- 속성값

  - visible : 영역을 벗어난 부분을 표시 (기본값)
 
  - hidden : 영역을 벗어난 부분을 잘라내어 보이지 않게 한다.
  ``` css
   .hidden {
      overflow: hidden;
    }
  ```
  - scroll : 영역을 벗어난 부분이 없어도 스크롤을 표시한다.
  ``` css
    .scroll {
      overflow: scroll;
    }
  ```
  - auto : 영역을 벗어난 부분이 있을때만 스크롤을 표시한다.
  ``` css
  .auto {
      overflow: auto;
  }
  ```
**특정 방향으로만 스크롤을 표시하고자 할 때는 overflow-x 또는 overflow-y 속성을 사용한다.**
``` css
.overflow-y {
    overflow-y: scroll;
}
```

### CSS float 

이 속성은 주로 레이아웃을 구성할 때 블록 레벨 요소를 가로 정렬하기 위해 사용하는 중요한 기법니다.

원래는 이미지와 텍스트가 같이 있을때 이미지 주위를 텍스트로 감싸기 위해 만들어진 것이다.

float 속성은 해당 요소를 다음 요소 위에 떠 있게 한다. 여기서 떠 있다는 의미는 요소가 기본 레이아웃 

흐름에서 벗어나 요소의 모서리가 페이지의 왼쪽이나 오른쪽에 이동하는 것이다. 

이 속성을 사용할때는 요소의 위치를 고정시키는 position 속성의 absolute를 사용하면 안된다. 

### 정렬

float 속성을 사용하지 않은 블록 요소들은 수직으로 정렬된다. 

float:left 속성을 사용하면 왼쪽부터 가로 정렬이 되고

float:right 속성을 사용하면 오른쪽부터 가로 정렬된다.

오른쪽 가로 정렬인 경우, 먼저 기술된 요소가 가장 오른쪽에 출력되므로 출력 순서가 역순이 된다.

``` css
.box.left {
  float:left;
}
```
``` css
.box.right {
  float: right;
}
```
