## 🖼️ img
- 이미지 표시
- src : 이미지 주소 입력
- alt : 대체 테스트
  - 그림이 깨졌을 때 보여주는 글씨
  - 웹 접근성에 따라 그림을 설명하는 내용을 적는다.
- width, height을 써서 넓이와 높이를 조절할 수 있다. 

## 📹 video
- 동영상 표시
- src : 동영상 주소 입력
- controls : 동영상 재생 도구 표시 태그
- loop : 동영상 반복 재생 태그
- width, height를 써서 너비와 높이를 조절할 수 있다.

## form
- form은 한 페이지에 여러 개 있을 수 있지만, form안에 form이 있으면 동작이 이상해질 수 있으니 넣지 않는것이 좋음
- 서버로 전달할 내용을 감싸는 태그.
- 속성
  - method
      - get, post (대소문자 구분 없음)
      - 기본값 : get
      - get (쿼리스트링 query string)
        - 주소에 name=value 형태로 서버에 전달
        - 여러개인 경우 &로 구분
            - name1=v1&name2=v2
        - 서버 주소 뒤에 ?으로 시작
      - post
        - 주소는 변경이 없음
        - body 영역에 담아서 서버로 전달
  - action
      - 전달할 서버의 주소
      - 기본값 : 현재 주소
   
## input

  - text
    - type속성의 기본값
    - 글씨 입력 받는 용도
    - 속성
      - vaule : 초기값
      - placeholder : 입력된 글씨가 없을 때 표시되는 글씨
      - readonly : 읽기 전용 (수정 불가)
      - disabled : 사용 금지

  - password : 글씨가 **** 이렇게 표시 됨.
  
  - button
      - 버튼
      - vaule 속성의 값으로 표시 글자를 적는다.

  - submit
      - form의 action 값으로 form 요소를 전송한다.
      - vaule 속성의 값으로 표시 글씨를 적는다.

  - checkbox
      - 0개 또는 1개 이상의 값을 선택할 수 있게 한다.
      - 선택된 것만 전송된다.
      - value의 값이 전송된다.
    <img width="604" height="71" alt="image" src="https://github.com/user-attachments/assets/a54a0e5b-9d78-4973-adbc-c4f72b36ac5b" />
      
      - 속성
          - checked : 체크가 기본으로 활성화 된 상태
          - disbled : 비활성화 

  - radio
    - 사용자가 1개만 선택할 수 있게 한다.
    - name 속성의 값이 같은 것끼리 그룹화 된다.
    - 속성
        - checked : 체크가 기본으로 활성화 된 상태.
        - disabled : 비활성화
     
  - select
    - select로 감싼 option을 선택하는 인터페이스
    - option으로 감싼 글씨가 표시된다.
    - 전송은 option의 value만 전송된다.
    - 속성
      - multiple : 한번에 여러 개 선택 가능
      - size : 한번에 보여줄 갯수 지정
     
  - textarea
    - 여러 줄의 글을 쓸 수 있다.
    - 태그 사이에 초기값을 정할 수 있다.
    - 속성
      - rows : 높이
      - cols : 너비
     
  - label
    - 선택 영역 확장 (checkbox, radio)
    
    - 방법은 총 2가지이며, 그 중에서 원하는 방법으로 사용하면 됨

    - input type="button"과 비슷하다.
    - 기본값 : type="button"
      - 그래서 꼭 type="button"을 적도록 하자.
    - 태그 사이에 버튼의 이름을 적는다.
