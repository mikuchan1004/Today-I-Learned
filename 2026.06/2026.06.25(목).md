## CSS (Cascading Style Sheets) 

HTML이나 XML과 같은 구조화 된 문서를 화면, 종이 등에 어떻게 렌더링할 것인지를 정의하기 위한 언어.

 **1. 셀렉터 (선택자) : 스타일을 적용하고자 하는 HTML 요소를 선택할 때 사용.**

   1-1. 전체 선택자 (*) : 현재 페이지에 있는 모든 태그를 선택.

   1-2. 태그 선택자 : 모든 태그에 적용.

   <img width="650" height="96" alt="image" src="https://github.com/user-attachments/assets/6b931402-161c-4ba5-9d8d-8fa4c30d80a0" />
  
   1-3. 다중 선택자 : 여러 선택자에 같은 내용을 한번에 적용. 선택자들을 컴마(,)로 나열
  
   <img width="649" height="109" alt="image" src="https://github.com/user-attachments/assets/8daf0f62-2c9b-4a00-903e-b04e6897c92d" />
  
   1-4. class 선택자 : 클래스명 앞에 .(점)을 찍는다.
  
   <img width="646" height="147" alt="image" src="https://github.com/user-attachments/assets/46e31007-7b05-4c74-b9eb-f8d4c96cea3b" />
  
    - class 속성은 모든 태그에 사용 가능
  
    - class 값은 가지는 모든 곳에 적용 됨
  
    - 중복을 허용.

    - 여러 태그에 같은 class 값 사용 가능
  
    - 띄어쓰기를 이용해서 여러개의 class를 가질 수 있다.

   1-5. id 선택자 : id 앞에 #를 적는다.
  
   <img width="633" height="136" alt="image" src="https://github.com/user-attachments/assets/c51147e5-b65f-44fe-8bec-d679ccaecbcd" />
    
    - id 태그는 모든 태그에 적용 가능.
    
    - 가능하면 한 페이지에는 하나의 id만 적용하는 것이 기본.
    
    - 값은 하나만 가질 수 있다.

   1-6. 속성 선택자

    - [속성명] : 해당 속성이 있는 모든 태그
    
    - [속성명 = 값] : 속성이 존재하고 값도 일치하는 모든 태그
    
    - [속성명^= 값] : 속성이 존재하고 값으로 시작하는 모든 태그
    
    - [속성명$ = 값] : 속성이 존재하고 값으로 끝나는 모든 태그

   <img width="643" height="95" alt="image" src="https://github.com/user-attachments/assets/504dca0a-da8b-4bdc-9035-c1f094fb7df0" />

    - hover : 마우스 올리면 적용
    
    - active : 마우스 클릭 시 적용

    - + : 바로 다음 형제를 선택

    - ~ : 이후에 나오는 모든 형제들을 선택

 **2. 프로퍼티 (속성) : 셀렉터로 HTML 요소를 선택하고 중괄호안에 속성과, 값을 지정하는 것으로 다양한 스타일 정의 가능.**

 **3. 값 (속성값) : 속성의 값은 해당 속성에 사용할 수 있는 값을 키워드나 크기 단위 또는 색상 표현 단위 등의 특정 단위로 지정해야만 적용이 됨.**

   예시. color : red ; / font-size : 16px; 또는 font-size : 300%;

 **4. HTML과 CSS의 연동 : HTML는 CSS를 포함할 수 있으며. CSS와 HTML을 연동하는 방법은 3가지가 있다.**

    4-1. 외장 스타일 시트 (External Style Sheet) : HTML에서 외부에 있는 CSS 파일을 로드하는 방식.

    4-2. 내장 스타일 시트 (Internal Style Sheet) : HTML 내부에 CSS를 포함시키는 방식.

    4-3. 인라인 스타일 (Inline Style) : HTML 요소의 스타일 속성에 CSS를 포함시키는 방식.

 **5. 우선 순위**

    - 구체적일수록 우선 순위가 높다.

    - 동일한 우선 순위인 경우 나중에 나온 것이 높다.
