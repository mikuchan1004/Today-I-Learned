## CSS 속성 값의 단위 

CSS 속성에는 키워드, 크기 단위, 색상 표현 단위 등의 특정 단위를 갖는 값을 지정한다.

1. 키워드 : 각 속성에 따라 사용할 수 있는 키워드가 존재한다.
  
  예시) display 속성의 값으로 사용할 수 있는 키워드는 block, inline, inline-block, none이 있다.

2. 크기 단위 : cm, mm, inch 등의 단위도 존재하지만 CSS에서 사용하는 대표적인 크기 단위는 px, em, %이다.
    
  - px는 절대값이고, em, %는 상대값.

  - 대부분 브라우저의 폰트 사이즈 기본값은 16px, 1em, 100%이며, 속성 값이 0일 경우, 단위를 생략할 수 있다.

2.1 px

  - 픽셀(화소)의 단위
  
  - 1px는 화소 1개 크기를 의미
  
  - 디바이스 해상도에 따라 상대적인 크기를 가짐.
  
  - 요소의 크기나 이미지의 크기 지정에 주로 사용됨.

## CSS 박스 모델 

모든 HTML 요소는 박스 형태의 영역을 가지고 있다.

  - 박스는 콘텐트(Content), 패딩(Padding), 테두리(Border), 마진(Margin)으로 구성.

  - 브라우저는 박스 모델의 크기와 속성, 위치를 근거로 해서 렌더링을 실행.

  - 콘텐트 : 요소의 텍스트나 이미지 등의 실제 내용이 위치하는 영역. width(너비), height(높이) 속성을 가진다.

  - 패딩 : 테두리 안쪽에 위치하는 요소의 내부 여부 영역. 패딩 속성 값은 패딩 영역의 두께를 의미. 기본색은 투명.

    요소에 적용된 배경의 컬러 이미지는 패딩 영역까지 적용됨.

    <img width="200" height="182" alt="image" src="https://github.com/user-attachments/assets/9faf03bf-d540-4bd6-b7ca-13dfba8e4f3d" />

  - 보더 : 테두리 영역으로 보더 속성 값은 테두리의 두께를 의미.

    <img width="254" height="182" alt="image" src="https://github.com/user-attachments/assets/1b8e66ef-f416-4fa8-be70-97519d63ecc7" />

  - 마진 : 테두리 바깥에 위치하는 요소의 외부 여백 영역. 마진 속성 값은 마진 영역의 두께를 의미. 기본적으로 투명하며 배경색은 지정할 수 없음.
