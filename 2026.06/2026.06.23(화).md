- a태그 : 앵커(anchor)의 약자. 연결해주는 태그. 브라우저 주소를 대신 적어주는 기능.
- a태그에 이미지, 텍스트, 컴퓨터에 저장된 파일도 넣을 수 있음. 페이지 연결, 페이지안에서 이동도 가능.

- 상대 주소 : 내 위치를 기준으로 삼는 주소. 현재 주소의 가장 마지막 / 이후의 주소를 변경한다.

- 절대 주소
1. 프로토콜로 시작 (http:// , https:// , ftp://)
2. // 로 시작 (프로토콜의 마지막 //)
3. / 로 시작
  3.1 ip:port/ <<< 여기를 뜻함

* 절대 주소가 아닌 모든 주소는 상대 주소를 뜻함
