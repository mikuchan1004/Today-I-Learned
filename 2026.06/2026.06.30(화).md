### CSS visibility 속성

visibility 속성은 요소를 보이게 할 것인지 보이지 않게 할 것인지를 정의.

- 속성 키워드

  - visible : 해당 요소를 보이게 한다 (기본값)
 
  - hidden : 해당 요소를 보이지 않게 한다
 
  - collapse : table 요소에 사용하며 행이나 열을 보이지 않게 한다.
 
  - none : table 요소의 row나 column을 보이지 않게 한다.

### CSS Opacity 속성 

요소의 투명도를 정의. 0.0 ~ 1.0의 값을 입력하며 0.0은 투명, 1.0은 불투명을 의미

### CSS visibility, Opacity HTML 코드 
``` html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>opacity</title>
    <style>
        .hidden {
            visibility: hidden;
        }

        .none {
            display:none;
        }

        .opacity {
            /* opacity: 0.3; */
            opacity:0;
        }
    </style>
</head>
<body>
    <p>첫번째</p>
    <p class="hidden">두번째</p>
    <p class="none">세번째</p>
    <p class="opacity">네번째</p>
    <p>다섯번째</p>
</body>
</html>
```

### CSS Background : 해당 요소의 배경으로 이미지 또는 색상을 정의.

- background-image : 요소에 배경 이미지를 지정

``` html
<html>
<head>
  <style>
    body {
      background-image: url("http://poiemaweb.com/img/bg/dot.png");
    }
  </style>
</head>
```
- background-repeat 속성 : 배경 이미지의 반복을 지정. 수직, 수평 또는 수직과 수평 모두의 반복을 지정할 수 있다.

  - repeat-x : X축으로만 배경이미지 반복
    ``` css
        .bg {
            background-repeat: repeat-x; /* x축으로만 배경이미지 반복 출력 */
        }
    ```
  - repeat-y : Y축으로만 배경이미지 반복
    ``` css
        .bg {
            background-repeat: repeat-y; /* y축으로만 배경이미지 반복 출력 */
        }
    ```
  - no-repeat : 반복 출력을 멈춤
    ``` css
        .bg {
            background-repeat: repeat-x; /* 배경이미지 반복 출력을 멈춘다. */
        }
    ```
 - background-size 속성 : 배경 이미지의 사이즈를 지정한다.

    - 배경 이미지의 고유 비율을 유지하기 때문에 설정에 따라 이미지의 일부가 보이지 않을 수 있다. 

    - 속성값은 px, %, cover, contain 등을 사용.
 
      - px 값 지정 : 배경이미지 크기가 지정된 px값 그대로 설정된다. 첫번째 값은 너비, 두번째 값은 높이를 의미.
     
        ``` css
        .bg {
            background-size: 700px 500px;
        }
        ```
      - % 값 지정 : 배경이미지 크기가 지정된 %값에 비례하여 설정된다. 첫번째 값은 너비, 두번째 값은 높이를 의미.

        화면을 줄이거나 늘리면 배경이미지의 크기도 따라서 변경되어 찌그러지는 현상이 나타난다.

        ``` css
        .bg {
            background-size: 100% 100%;
        }
        ```
      - cover 지정 : 배경이미지의 크기 비율을 유지한 상태에서 부모 요소의 너비, 높이 중 큰값에 배경이미지를 맞춘다.

        이미지의 일부가 보이지 않을 수 있다.

        ``` css
        .bg {
            background-size: cover;
        }
        ```
      - contain 지정 : 배경이지의 크기 비율을 유지한 상태에서 부모 요소의 영역에 배경이미지가
      
        보이지 않는 부분없이 전체가 들어갈 수 있도록 이미지 스케일을 조정한다.

        ``` css
        .bg {
            background-size: contain;
        }
        ```
- background-attachment 속성

일반적으로 화면을 스크롤하면 배경 이미지도 함께 스크롤되는데, 화면이 스크롤되더라도 

배경이미지는 스크롤되지 않고 고정되어 있게 하려면 background-attachment 속성에 fixed 키워드를 지정하면 된다.

``` css
    .bg {
        background-attachment: fixed; /* 이렇게 지정해두면 화면이 스크롤되더라도 배경이미지는 스크롤되지 않는다 */
    }
```
- background-position 속성

일반적으로 background-image는 좌상단부터 이미지를 출력하는데, 

background-position 속성을 사용하면 이미지의 좌표를 지정할 수 있다. 

기본값은 background-position: 0% 0%; 로 배경이미지는 우측 상단에 위치하게 된다.

``` css
    .bg {
        background-position: 0% 0%; /* 배경이미지는 우측 상단에 위치 */
    }
```
